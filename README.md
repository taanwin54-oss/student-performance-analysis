# Student Performance Analysis

This project analyzes student performance using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Analysis Performed
- Data cleaning
- Average grade calculation
- Gender performance comparison
- Study time vs grade correlation

## Visualizations
- Histogram of grades
- Scatter plot of study time vs grades
- Bar chart of gender vs average grade

## Conclusion
The analysis shows relationships between study time, gender, and final student performance.
